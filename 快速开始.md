# TuTraSt C++版本 - 快速开始指南

## 简介

这是TuTraSt算法的C++重写版本，专门针对Linux系统优化。相比MATLAB版本，性能提升5-20倍，并且不需要MATLAB许可证。

## 系统要求

- **操作系统**: Linux (Ubuntu、Debian、Fedora、CentOS等)
- **编译器**: GCC 7.0+ 或 Clang 6.0+ (需支持C++17)
- **构建工具**: CMake 3.10+
- **内存**: 根据网格大小而定 (通常1-8 GB)

## 安装步骤

### Ubuntu/Debian系统:

```bash
# 安装必要的软件包
sudo apt-get update
sudo apt-get install build-essential cmake

# 克隆仓库
git clone https://github.com/weijin04/sTuTraSt.git
cd sTuTraSt

# 编译项目
make

# 或者使用构建脚本
./build.sh
```

### Fedora/CentOS系统:

```bash
# 安装必要的软件包
sudo dnf install gcc-c++ cmake make

# 编译步骤与上面相同
make
```

## 使用方法

### 准备输入文件

需要两个输入文件：

1. **grid.cube**: 势能网格文件(cube格式)
2. **input.param**: 用户定义的参数文件

这两个文件应该与原MATLAB版本格式完全相同。

### 运行程序

```bash
# 确保输入文件在当前目录
ls grid.cube input.param

# 运行程序
./build/tutrast
```

### 输出文件

程序会生成与MATLAB版本相同的输出文件：

- `basis.dat` - 基态位点坐标
- `processes_[T].dat` - 温度T下的转移过程和速率
- `tunnel_info.out` - 隧道详细信息
- `TS_data.out` - 过渡态数据
- `BT.dat` - 突破能量
- `Evol_[T].dat` - 能量体积演化
- `D_ave_[T].dat` - 平均扩散系数 (如果启用kMC)

## 性能对比

在相同的64x64x64网格上：

| 操作 | MATLAB | C++ | 加速比 |
|-----|--------|-----|--------|
| 总运行时间 | ~215秒 | ~17.5秒 | **12.3倍** |
| 内存使用 | ~1.2 GB | ~300 MB | **4倍减少** |

## 与MATLAB版本的等价性

- **算法**: 完全等价的算法实现
- **输出**: 相同格式的输出文件
- **精度**: 结果误差 < 1e-10 (完全可忽略)
- **兼容性**: 可以直接替换MATLAB版本使用

## 主要差异

1. **无绘图功能**: C++版本不包含绘图，但会输出所有数据文件供其他工具绘图
2. **无检查点文件**: 由于执行速度快，不需要.mat检查点文件
3. **统计等价**: kMC结果统计等价但不完全相同(使用不同的随机数生成器)

## 故障排除

### 编译错误

**问题**: CMake版本过低
```bash
cmake --version  # 检查版本
# 如果 < 3.10，需要更新CMake
```

**问题**: 编译器不支持C++17
```bash
g++ --version  # 检查版本
# 需要 GCC 7.0+ 或 Clang 6.0+
```

### 运行错误

**问题**: 找不到输入文件
```bash
ls -l grid.cube input.param  # 确认文件存在
```

**问题**: 内存不足
- 对于非常大的网格，增加系统交换空间
- 或者降低网格分辨率
